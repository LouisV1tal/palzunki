# Ucell BNPL Calculator

Интерактивная финансовая модель: снижение цены на телефоны + апгрейд тарифа.

## Запуск локально

```bash
npm install
npm start
```

Откроется на http://localhost:3000

## Деплой на GitHub Pages

```bash
# 1. Установить gh-pages
npm install --save-dev gh-pages

# 2. Добавить в package.json:
#    "homepage": "https://ВАШ_ЮЗЕР.github.io/ucell-calculator"
#    "predeploy": "npm run build"
#    "deploy": "gh-pages -d build"

# 3. Задеплоить
npm run deploy
```

## Что считает модель

- **Потери на телефонах** — Ucell снижает цену ниже самого дешёвого конкурента на 4%, теряет разницу
- **Тарифный профит** — апгрейднутые клиенты платят повышенный тариф × маржа × 12 месяцев
- **Окупаемость** — через сколько месяцев накопленный профит перекрывает потери
- **Стабильный профит** — когда ежемесячная прибыль превышает ежемесячные потери

## Источники данных

- Данные продаж: реестр Ucell (605 BNPL-продаж за 1.5-месячный цикл)
- Цены конкурентов: BeeMarket, MediaPark, Texnomart, AlifShop, Idea
- Тарифы Ucell: Bor 70/90/110/160, TV Bor, TV Doimiy, Ovoz, Foydali и др.
- Маржинальность Ucell на тарифах: 75% (из модели Витек.xlsx)
